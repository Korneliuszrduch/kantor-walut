## How does it work?
In the first field, enter the amount in Polish zloty, and in the second field, select the currency you want to convert to. Then click 'Convert' and get the results.
## Used features
1. HTML
2. BEM convention
3. CSS
4. Flex model
5. JavaScript (ES6+)
